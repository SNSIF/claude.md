# 12-1. SDK 기본

## Anthropic SDK 개요

Anthropic SDK는 Claude API를 애플리케이션에서 쉽게 사용하기 위한 공식 클라이언트 라이브러리입니다. HTTP 요청을 직접 작성할 필요 없이, 간결한 코드로 Claude와 통신할 수 있습니다.

**SDK가 제공하는 기능:**
- 인증 처리 (API 키 관리)
- 요청/응답 직렬화 및 역직렬화
- 자동 재시도 및 에러 처리
- 타입 안전성 (TypeScript/Python 타입 힌트)
- 스트리밍 지원

## API 키 설정

Anthropic API 키는 https://console.anthropic.com 에서 발급받을 수 있습니다.

### 환경 변수로 설정 (권장)

```bash
# macOS / Linux
export ANTHROPIC_API_KEY="sk-ant-..."

# Windows (Command Prompt)
set ANTHROPIC_API_KEY=sk-ant-...

# Windows (PowerShell)
$env:ANTHROPIC_API_KEY = "sk-ant-..."
```

`.bashrc` 또는 `.zshrc`에 추가하면 영구 설정됩니다.

```bash
echo 'export ANTHROPIC_API_KEY="sk-ant-..."' >> ~/.bashrc
source ~/.bashrc
```

### 코드에서 직접 전달 (비권장 — 키 유출 주의)

```python
# 절대 코드에 하드코딩하지 말 것. 환경 변수 사용을 권장함.
client = anthropic.Anthropic(api_key="sk-ant-...")
```

## 기본 사용법 — Python

```python
import anthropic

# API 키는 환경 변수 ANTHROPIC_API_KEY에서 자동으로 읽음
client = anthropic.Anthropic()

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "안녕하세요!"}
    ]
)

print(message.content[0].text)
# 출력: 안녕하세요! 저는 Claude입니다. 무엇을 도와드릴까요?
```

### 시스템 프롬프트 추가

## 기본 사용법 — TypeScript

```typescript
import Anthropic from "@anthropic-ai/sdk";

// API 키는 환경 변수 ANTHROPIC_API_KEY에서 자동으로 읽음
const client = new Anthropic();

const message = await client.messages.create({
    model: "claude-sonnet-4-6",
    max_tokens: 1024,
    messages: [{ role: "user", content: "안녕하세요!" }]
});

console.log(message.content[0].text);
// 출력: 안녕하세요! 저는 Claude입니다. 무엇을 도와드릴까요?
```

### async/await 전체 예제

```typescript
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();

async function main() {
    const message = await client.messages.create({
        model: "claude-sonnet-4-6",

            }

main().catch(console.error);
```

## 최신 모델 ID

| 모델 | 모델 ID | 특징 |
|------|---------|------|
| Claude Opus 4.6 | `claude-opus-4-6` | 최고 성능, 복잡한 작업 |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | 성능과 속도의 균형 |
| Claude Haiku 4.5 | `claude-haiku-4-5-20251001` | 빠르고 저렴, 간단한 작업 |

> 모델 선택 기준: 빠른 응답이 필요하면 Haiku, 고품질 응답이 필요하면 Sonnet, 극도로 복잡한 추론이 필요하면 Opus

## 주요 파라미터

```python
response = client.messages.create(
    # 필수 파라미터
    model="claude-sonnet-4-6",       # 사용할 모델 ID
    max_tokens=1024,                  # 최대 출력 토큰 수 (비용과 직결)
    messages=[...],                   # 대화 이력 (role + content 목록)

    # 선택 파라미터
    system="시스템 프롬프트",          # Claude의 역할/행동 지침
    temperature=0.7,                  # 창의성 조절 (0=결정적, 1=창의적)
    top_p=0.9,                        # 토큰 샘플링 범위 (temperature와 함께 잘 쓰지 않음)
    stop_sequences=["END", "STOP"],   # 이 문자열이 나오면 생성 중단
    metadata={"user_id": "u123"},     # 사용자 식별 메타데이터 (선택)
)
```

### temperature 가이드

| temperature | 사용 사례 |
|------------|---------|
| 0.0 | 코드 생성, 데이터 추출 (일관성 최우선) |
| 0.3 ~ 0.5 | 분석, 요약, Q&A |
| 0.7 ~ 1.0 | 창의적 글쓰기, 브레인스토밍 |

## 응답 구조

`client.messages.create()`의 반환값은 `Message` 객체입니다.

```python
response = client.messages.create(...)

print(response.id)              # msg_xxxxxxxxxx — 메시지 고유 ID
print(response.type)            # "message"
print(response.role)            # "assistant"
print(response.model)           # 실제 사용된 모델 ID
print(response.stop_reason)     # "end_turn" | "max_tokens" | "tool_use" | "stop_sequence"

# 콘텐츠 블록 목록
for block in response.content:
    if block.type == "text":
        print(block.text)

# 토큰 사용량
print(response.usage.input_tokens)   # 입력 토큰 수
print(response.usage.output_tokens)  # 출력 토큰 수
```

## 멀티턴 대화 구현

Claude는 대화 이력을 직접 관리합니다. `messages` 배열에 이전 대화를 모두 포함해야 합니다.

```python
import anthropic

client = anthropic.Anthropic()

def chat():
    messages = []
    print("Claude와 대화를 시작합니다. 'quit'을 입력하면 종료됩니다.\n")

    while True:
        user_input = input("나: ")
        if user_input.lower() == "quit":
            break

        # 사용자 메시지 추가
        messages.append({"role": "user", "content": user_input})

        # Claude에게 요청
        response = client.messages.create(
            model="claude-sonnet-4-6",
            max_tokens=1024,
            messages=messages  # 전체 대화 이력 전달
        )

        assistant_message = response.content[0].text

        # 어시스턴트 응답을 이력에 추가
        messages.append({"role": "assistant", "content": assistant_message})

        print(f"Claude: {assistant_message}\n")

chat()
```

**실행 예시:**
```
나: 내 이름은 김철수야
Claude: 안녕하세요, 김철수 님! 반갑습니다.

나: 내 이름이 뭐야?
Claude: 방금 말씀해주셨듯이 김철수 님이시죠!
```

## 에러 처리

```python
import anthropic
from anthropic import APIError, RateLimitError, AuthenticationError

client = anthropic.Anthropic()

try:
    response = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=1024,
        messages=[{"role": "user", "content": "안녕!"}]
    )
    print(response.content[0].text)

except AuthenticationError:
    print("API 키가 잘못되었습니다. ANTHROPIC_API_KEY를 확인하세요.")
except RateLimitError:
    print("요청 한도 초과. 잠시 후 다시 시도하세요.")
except APIError as e:
    print(f"API 오류 발생: {e.status_code} - {e.message}")
```

## 실습: 간단한 챗봇 만들기

아래 코드를 `chatbot.py`로 저장하고 실행해보세요.

```python
import anthropic
import os

def main():
    # API 키 확인
    if not os.environ.get("ANTHROPIC_API_KEY"):
        print("오류: ANTHROPIC_API_KEY 환경 변수가 설정되지 않았습니다.")
        print("실행: export ANTHROPIC_API_KEY='sk-ant-...'")
        return

    client = anthropic.Anthropic()
    messages = []

    # 챗봇 성격 설정
    system_prompt = """당신은 친절하고 유머 있는 AI 어시스턴트입니다.
한국어로 대화하며, 답변은 간결하게 2~3문장으로 합니다."""

    print("=" * 40)
    print("  Claude 챗봇에 오신 것을 환영합니다!")
    print("  종료하려면 'quit' 또는 'q'를 입력하세요.")
    print("=" * 40)
    print()

    while True:
        try:
            user_input = input("나: ").strip()
        except KeyboardInterrupt:
            print("\n대화를 종료합니다.")
            break

        if not user_input:
            continue
        if user_input.lower() in ("quit", "q", "exit"):
            print("대화를 종료합니다. 안녕히 계세요!")
            break

        messages.append({"role": "user", "content": user_input})

        try:
            response = client.messages.create(
                model="claude-sonnet-4-6",
                max_tokens=512,
                system=system_prompt,
                messages=messages
            )
            reply = response.content[0].text
            messages.append({"role": "assistant", "content": reply})
            print(f"Claude: {reply}")
            print()

        except Exception as e:
            print(f"오류: {e}")
            messages.pop()  # 실패한 메시지 제거

if __name__ == "__main__":
    main()
```

**실행 방법:**
```bash
export ANTHROPIC_API_KEY="sk-ant-..."
python chatbot.py
```
