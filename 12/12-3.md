# 12-3. Streaming (스트리밍)

## Streaming이란?

스트리밍(Streaming)은 Claude의 응답을 생성되는 즉시 토큰 단위로 실시간 수신하는 방식입니다. 기본 방식(non-streaming)은 Claude가 응답을 모두 생성한 뒤 한꺼번에 반환하지만, 스트리밍은 타이핑하듯 글자가 하나씩 전달됩니다.

---

## 왜 스트리밍이 필요한가?

**1. UX 개선 — 첫 토큰까지 대기 시간 최소화**

긴 답변을 생성하는 경우 non-streaming 방식은 수 초 동안 화면이 비어 있습니다. 스트리밍을 사용하면 첫 단어가 즉시 나타나므로 사용자가 "작동 중"임을 바로 알 수 있습니다.

**2. 긴 응답을 점진적으로 표시**

보고서나 코드처럼 긴 콘텐츠를 생성할 때 사용자가 생성이 완료되기 전에 내용을 미리 읽기 시작할 수 있습니다.

**3. 진행 상황 실시간 모니터링**

에이전트나 배치 작업에서 처리 중인 내용을 실시간으로 확인할 수 있습니다.

---

## Python 스트리밍 — context manager 방식 (권장)

```python
import anthropic

client = anthropic.Anthropic()

with client.messages.stream(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[{"role": "user", "content": "한국의 역사에 대해 자세히 설명해줘"}]
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

    # 스트림 완료 후 전체 메시지 접근
    final_message = stream.get_final_message()
    print(f"\n\n토큰 사용: {final_message.usage}")
```

`flush=True`는 버퍼링 없이 즉시 출력하기 위해 필요합니다.

## TypeScript 스트리밍

```typescript
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();

const stream = client.messages.stream({
    model: "claude-sonnet-4-6",
    max_tokens: 1024,
    messages: [{ role: "user", content: "한국의 역사에 대해 설명해줘" }]
});

// 텍스트 이벤트 리스너
stream.on("text", (text) => {
    process.stdout.write(text);
});

// 스트림 완료 대기
const finalMessage = await stream.finalMessage();
console.log("\n완료:", finalMessage.usage);
```

### TypeScript — async iterator 방식

```typescript
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();

async function streamResponse(prompt: string): Promise<void> {
    const stream = await client.messages.create({
        model: "claude-sonnet-4-6",
        max_tokens: 1024,
        stream: true,
        messages: [{ role: "user", content: prompt }]
    });

    for await (const event of stream) {
        if (
            event.type === "content_block_delta" &&
            event.delta.type === "text_delta"
        ) {
            process.stdout.write(event.delta.text);
        }
    }
    console.log(); // 줄바꿈
}

await streamResponse("JavaScript의 클로저를 설명해줘");
```

## 스트리밍 이벤트 타입

스트림에서 발생하는 이벤트의 순서와 의미입니다.

| 이벤트 타입 | 발생 시점 | 주요 데이터 |
|------------|---------|-----------|
| `message_start` | 스트림 시작 | 메시지 ID, 모델명, 초기 usage |
| `content_block_start` | 새 콘텐츠 블록 시작 | 블록 인덱스, 블록 타입(text/tool_use) |
| `content_block_delta` | 텍스트 조각 수신 | `text_delta` 또는 `input_json_delta` |
| `content_block_stop` | 현재 콘텐츠 블록 종료 | - |
| `message_delta` | 메시지 수준 변경 | `stop_reason`, 누적 usage |
| `message_stop` | 스트림 완전 종료 | - |

```python
# 모든 이벤트를 출력해보는 예제
import anthropic

client = anthropic.Anthropic()

with client.messages.stream(
    model="claude-sonnet-4-6",
    max_tokens=100,
    messages=[{"role": "user", "content": "안녕"}]
) as stream:
    for event in stream:
        print(f"이벤트: {event.type}")
```

## Tool Use + 스트리밍 조합

도구 호출과 스트리밍을 함께 사용할 수 있습니다. 이 경우 도구 입력도 점진적으로 스트리밍됩니다.

```python
import anthropic
import json

client = anthropic.Anthropic()

tools = [
    {
        "name": "get_weather",
        "description": "도시의 날씨를 반환합니다",
        "input_schema": {
            "type": "object",
            "properties": {
                "city": {"type": "string", "description": "도시 이름"}
            },
            "required": ["city"]
        }
    }
]

def get_weather(city: str) -> dict:
    return {"city": city, "temperature": "22°C", "condition": "맑음"}

messages = [{"role": "user", "content": "서울 날씨 알려줘"}]

# 스트리밍으로 첫 번째 응답
with client.messages.stream(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    tools=tools,
    messages=messages
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

    final = stream.get_final_message()

# 도구 호출이 있는 경우 처리
if final.stop_reason == "tool_use":
    messages.append({"role": "assistant", "content": final.content})
    tool_results = []

    for block in final.content:
        if block.type == "tool_use":
            result = get_weather(**block.input)
            tool_results.append({
                "type": "tool_result",
                "tool_use_id": block.id,
                "content": json.dumps(result, ensure_ascii=False)
            })

    messages.append({"role": "user", "content": tool_results})

    # 최종 응답도 스트리밍
    print("\n[최종 응답]")
    with client.messages.stream(
        model="claude-sonnet-4-6",
        max_tokens=1024,
        tools=tools,
        messages=messages
    ) as stream:
        for text in stream.text_stream:
            print(text, end="", flush=True)
```

## 에러 처리

스트리밍 도중 네트워크 오류나 API 오류가 발생할 수 있습니다.

```python
import anthropic
from anthropic import APIConnectionError, RateLimitError, APIStatusError

client = anthropic.Anthropic()

def stream_with_retry(prompt: str, max_retries: int = 3) -> str:
    """재시도 로직이 포함된 스트리밍 함수"""
    for attempt in range(max_retries):
        try:
            result = []
            with client.messages.stream(
                model="claude-sonnet-4-6",
                max_tokens=1024,
                messages=[{"role": "user", "content": prompt}]
            ) as stream:
                for text in stream.text_stream:
                    print(text, end="", flush=True)
                    result.append(text)
            print()
            return "".join(result)

        except APIConnectionError as e:
            print(f"\n연결 오류 (시도 {attempt + 1}/{max_retries}): {e}")
            if attempt < max_retries - 1:
                import time
                time.sleep(2 ** attempt)  # 지수 백오프
        except RateLimitError:
            print("\n요청 한도 초과. 60초 후 재시도...")
            import time
            time.sleep(60)
        except APIStatusError as e:
            print(f"\nAPI 오류 {e.status_code}: {e.message}")
            break

    return ""

stream_with_retry("파이썬의 장점을 설명해줘")
```

## 실습: 실시간 타이핑 효과 챗봇 만들기

`streaming_chatbot.py`로 저장하고 실행해보세요.

```python
import anthropic
import sys
import time

client = anthropic.Anthropic()

def typing_effect_chatbot():
    """실시간 타이핑 효과가 있는 챗봇"""
    messages = []
    system_prompt = "당신은 유용한 AI 어시스턴트입니다. 한국어로 친절하게 답변해주세요."

    print("=" * 50)
    print("  스트리밍 챗봇 (실시간 타이핑 효과)")
    print("  'quit'을 입력하면 종료됩니다.")
    print("=" * 50)
    print()

    while True:
        try:
            user_input = input("나: ").strip()
        except KeyboardInterrupt:
            print("\n\n종료합니다.")
            break

        if not user_input:
            continue
        if user_input.lower() in ("quit", "q"):
            print("안녕히 계세요!")
            break

        messages.append({"role": "user", "content": user_input})

        print("Claude: ", end="", flush=True)
        full_response = []

        try:
            with client.messages.stream(
                model="claude-sonnet-4-6",
                max_tokens=1024,
                system=system_prompt,
                messages=messages
            ) as stream:
                for text in stream.text_stream:
                    print(text, end="", flush=True)
                    full_response.append(text)

                final = stream.get_final_message()

            print(f"\n[입력: {final.usage.input_tokens}토큰 / 출력: {final.usage.output_tokens}토큰]\n")

            messages.append({
                "role": "assistant",
                "content": "".join(full_response)
            })

        except Exception as e:
            print(f"\n오류 발생: {e}")
            messages.pop()  # 실패한 메시지 제거

if __name__ == "__main__":
    typing_effect_chatbot()
```
