# 2-3. 권한 모드

## 권한 모드란?

Claude Code가 작업할 때 얼마나 자율적으로 움직일지를 결정하는 모드입니다.
VSCode에서는 채팅 입력창 옆 버튼이나 `/mode` 커맨드로 전환할 수 있습니다.

---

## 4가지 모드

### default
기본 모드. 파일 수정·명령 실행 등 잠재적으로 위험한 작업은 매번 승인을 요청합니다.

### acceptEdits
파일 읽기·쓰기·수정은 자동 승인, Bash 명령은 여전히 승인 요청합니다.
코드 작업 위주일 때 승인 피로를 줄여줍니다.

```
파일 작업 → 자동 승인
Bash 명령  → 승인 요청
```

### plan
실제 작업을 수행하지 않고 **계획만 세웁니다**.
무엇을 할지 먼저 확인하고 싶을 때 사용합니다. → [8-1. Plan Mode](../8/8-1.md)

### bypassPermissions
모든 작업을 승인 없이 자동 실행합니다.
**신뢰할 수 있는 환경(CI, 자동화 스크립트)에서만 사용합니다.**

---

## 모드 전환 방법

채팅에서 직접 지정:
```
/mode acceptEdits
```

또는 settings.json에서 기본값 지정:
```json
{
  "defaultMode": "acceptEdits"
}
```

---

## 권장 사용 패턴

| 상황 | 권장 모드 |
|------|-----------|
| 일반 개발 작업 | `acceptEdits` |
| 낯선 코드베이스 탐색 | `default` 또는 `plan` |
| 큰 리팩터링 전 계획 확인 | `plan` |
| CI / 자동화 | `bypassPermissions` |
