# 7-1. 자동 메모리 시스템

## 메모리란?

Claude Code가 대화 중에 학습한 내용을 **파일로 저장**해두는 시스템입니다.
다음 대화에서도 이전에 파악한 컨텍스트를 기억합니다.

---

## 저장 위치

```
~/.claude/projects/[프로젝트 경로]/memory/   # 프로젝트별 메모리
~/.claude/memory/                             # 전역 메모리
```

각 메모리는 Markdown 파일로 저장됩니다.

---

## 메모리 유형

| 유형 | 저장되는 내용 |
|------|---------------|
| `user` | 사용자의 역할, 선호도, 기술 수준 |
| `feedback` | "이렇게 하지 마", "이 방식이 좋았어" 같은 피드백 |
| `project` | 진행 중인 작업, 결정 사항, 마감일 |
| `reference` | 외부 시스템 위치 (Linear 프로젝트, Grafana 대시보드 등) |

---

## 자동 저장 시점

Claude가 다음을 감지하면 자동으로 저장합니다:

- 사용자의 역할·선호도 파악 ("나는 백엔드 위주야")
- 명시적 피드백 ("이런 식으로 하지 마", "딱 이거야")
- 프로젝트 결정 사항 ("이번 릴리즈는 금요일까지")

## 메모리 인덱스 파일

`MEMORY.md` 에 모든 메모리 파일의 목록이 인덱싱됩니다.
Claude는 대화 시작 시 이 파일을 읽어 어떤 메모리가 있는지 파악합니다.

메모리 관리 방법은 **[7-2. 메모리 관리](7-2.md)** 에서 다룹니다.
