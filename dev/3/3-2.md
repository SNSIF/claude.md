# 3-2. Write — 파일 생성

## 기본 사용

새 파일을 만들거나 파일 전체를 교체할 때 사용합니다.

```
src/utils/formatDate.ts 파일 만들어줘. 날짜를 YYYY-MM-DD 형식으로 변환하는 함수
```

---

## Write vs Edit

| 상황 | 적합한 도구 |
|------|-------------|
| 새 파일 생성 | Write |
| 기존 파일 일부 수정 | Edit |
| 파일 전체 재작성 | Write |

기존 파일의 특정 부분만 바꾸고 싶다면 **[3-3. Edit](3-3.md)** 를 사용합니다.

---

## 여러 파일 한번에

```
로그인 페이지 만들어줘.
- src/pages/Login.tsx
- src/hooks/useLogin.ts
- src/api/auth.ts
이 세 파일 생성해줘
```

Claude는 파일 간 의존성을 고려해 순서대로 생성합니다.

---

## 주의사항

- Write는 기존 파일을 **덮어씁니다**. 실수로 기존 내용이 날아갈 수 있습니다.
- Claude Code는 Write 실행 전 승인을 요청합니다 (acceptEdits 모드에서는 자동 승인).
- 중요한 파일은 git으로 관리해 롤백에 대비합니다.
