# 3-5. Grep — 내용 검색

## 기본 사용

파일 내용에서 특정 텍스트나 패턴을 찾습니다.

```
useAuth가 어디서 사용되고 있어?
환경변수 NEXT_PUBLIC_ 로 시작하는 게 어디서 쓰이는지 찾아줘
```

---

## 정규표현식 지원

```
TODO 주석이 전부 어디 있어?
console.log 가 남아있는 파일 찾아줘
```

내부적으로 `TODO.*`, `console\.log` 같은 정규표현식 패턴을 사용합니다.

---

## 자주 쓰는 검색 예시

```
// 특정 함수 사용 위치
"fetchUser 함수가 어디서 호출돼?"

// import 경로 추적
"@/hooks/useAuth 를 import하는 파일 전부 찾아줘"

// 하드코딩된 값
"localhost:3000 이 하드코딩된 곳 있어?"

// 사용되지 않는 코드
"DeprecatedComponent 가 아직 쓰이는 곳 있어?"
```

---

## 범위 제한

```
src/components/ 안에서만 Button을 쓰는 파일 찾아줘
```

디렉터리를 명시하면 검색 범위를 좁혀서 더 빠르게 찾습니다.

---

## Glob과의 차이

| 도구 | 기준 | 예시 |
|------|------|------|
| Grep | 파일 내용 | `useAuth` → 해당 텍스트가 있는 파일 |
| Glob | 파일 경로/이름 | `**/*.ts` → TypeScript 파일 목록 |
