# 3-1. Read — 파일 읽기

## 기본 사용

Claude에게 파일을 읽어달라고 하면 됩니다.

```
src/components/Button.tsx 읽어줘
```

내부적으로 Read 도구를 사용해 파일 전체를 읽습니다.

---

## 라인 범위 지정

큰 파일에서 특정 부분만 읽을 때:

```
src/api/auth.ts 50~100번째 줄 읽어줘
```

---

## 여러 파일 동시에

```
src/hooks/useAuth.ts 와 src/context/AuthContext.tsx 둘 다 읽어줘
```

Claude는 여러 파일을 병렬로 읽어 비교하거나 연관 분석을 합니다.

---

## 지원 파일 형식

| 형식 | 지원 여부 |
|------|-----------|
| 텍스트 (.ts, .py, .md 등) | ✅ |
| 이미지 (.png, .jpg 등) | ✅ (내용 분석 가능) |
| PDF | ✅ (텍스트 추출) |
| 바이너리 | ❌ |

---

## 실전 팁

**파일을 직접 언급하지 않아도 됩니다.**
"이 프로젝트에서 인증 처리가 어떻게 되어 있어?" 라고 물으면
Claude가 Glob/Grep으로 관련 파일을 찾아서 스스로 읽습니다.

**VSCode에서 @-mention 활용:**
채팅 입력창에서 `@파일명`으로 파일을 직접 첨부할 수 있습니다.
