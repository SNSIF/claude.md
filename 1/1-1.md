# 1-1. 설치 및 인증

## Claude Code란?

Claude Code는 Anthropic이 만든 공식 CLI(Command Line Interface) 도구로, 터미널에서 직접 Claude AI와 대화하며 코드 작성, 파일 수정, Git 작업, 프로젝트 관리 등을 수행할 수 있습니다.

**핵심 특징:**
- 파일 읽기/쓰기/편집, 코드 검색, 터미널 명령 실행 등의 도구를 Claude가 직접 호출
- 단순 채팅이 아닌 **에이전트 방식** — Claude가 스스로 판단하여 여러 도구를 조합
- 프로젝트 전체 맥락을 이해하고 작업
- CI/CD 파이프라인, 자동화 스크립트에도 통합 가능

## 플랫폼별 설치 방법

### macOS / Linux (npm)

```bash
npm install -g @anthropic-ai/claude-code
```

### Windows PowerShell

```powershell
npm install -g @anthropic-ai/claude-code
```

> Node.js 18 이상이 필요합니다. [nodejs.org](https://nodejs.org)에서 설치하세요.

### Windows CMD

```cmd
npm install -g @anthropic-ai/claude-code
```

### Homebrew (macOS/Linux)

```bash
brew install claude-code
```

### WinGet (Windows)

```powershell
winget install Anthropic.ClaudeCode
```

## 설치 확인

설치가 완료되면 버전을 확인하여 정상 설치 여부를 검증합니다.

```bash

```

출력 예시:
```

```

## 인증 방법

Claude Code를 사용하려면 Anthropic 계정으로 인증해야 합니다.

### 방법 1: 대화형 세션 내 슬래시 명령

```bash
claude        # 대화형 세션 시작
/login        # 세션 내에서 로그인
```

### 방법 2: CLI 직접 인증

```bash
claude auth login
```

위 명령을 실행하면 브라우저가 열리고 Anthropic 계정으로 OAuth 로그인을 진행합니다. 로그인 완료 후 터미널에 인증 성공 메시지가 표시됩니다.

### 방법 3: API 키 직접 설정 (자동화 환경)

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
claude -p "hello"
```

또는 `.env` 파일:

```env
ANTHROPIC_API_KEY=sk-ant-...
```

## 계정 유형 설명

| 계정 유형 | 설명 | 적합한 대상 |
|-----------|------|------------|
| **Claude Pro** | 월정액 구독, claude.ai 계정 사용 | 개인 개발자 |
| **Claude Max** | Pro보다 높은 사용량 한도 | 헤비 유저 |
| **Anthropic Console (API)** | API 크레딧 기반 과금 | 팀/기업, 자동화 |
| **AWS Bedrock** | AWS 인프라를 통한 접근 | AWS 사용 기업 |
| **Google Vertex AI** | GCP 인프라를 통한 접근 | GCP 사용 기업 |
| **Azure AI Foundry** | Azure 인프라를 통한 접근 | Azure 사용 기업 |

### Bedrock / Vertex / Foundry 설정 예시

```bash
# AWS Bedrock
export CLAUDE_CODE_USE_BEDROCK=1
export AWS_REGION=us-east-1

# Google Vertex AI
export CLAUDE_CODE_USE_VERTEX=1
export CLOUD_ML_REGION=us-east5
export ANTHROPIC_VERTEX_PROJECT_ID=my-project

claude auth login
```

## 트러블슈팅

### 문제: `command not found: claude`

npm의 글로벌 bin 경로가 PATH에 없는 경우입니다.

```bash
# npm 글로벌 경로 확인
npm config get prefix

# macOS/Linux — ~/.bashrc 또는 ~/.zshrc에 추가
export PATH="$PATH:$(npm config get prefix)/bin"

# 적용
source ~/.zshrc
```

### 문제: 인증 후에도 로그인 안 됨

```bash
# 인증 정보 초기화 후 재시도
claude auth logout
claude auth login
```

### 문제: Node.js 버전 오류

```bash
node --version   # v18 이상 필요
```

nvm을 사용하는 경우:

```bash
nvm install 20
nvm use 20
npm install -g @anthropic-ai/claude-code
```

### 문제: Windows에서 PowerShell 실행 정책 오류

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

### 문제: 프록시 환경에서 연결 실패

```bash
export HTTPS_PROXY=http://proxy.company.com:8080
claude auth login
```
