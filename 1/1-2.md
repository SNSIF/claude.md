# 1-2. 첫 대화

## 대화형 세션 시작

터미널에서 `claude` 명령만 입력하면 대화형 세션이 시작됩니다.

```bash
claude
```

세션이 시작되면 아래와 같은 프롬프트가 표시됩니다:

```
Claude Code v1.x.x
Type /help for available commands.

>
```

---

## 기본 인터페이스 설명

### 프롬프트 (`>`)
사용자 입력을 기다리는 상태입니다. 자연어로 무엇이든 입력할 수 있습니다.

```
> 현재 디렉토리의 파일 목록을 보여줘
```

### 응답 영역
Claude의 텍스트 응답이 출력됩니다. Markdown 형식으로 렌더링됩니다.

### 도구 사용 표시
Claude가 파일을 읽거나 명령을 실행할 때 아래처럼 표시됩니다:

```
[도구 사용] Bash: ls -la
[도구 사용] Read: src/main.py
[도구 사용] Write: output.txt
```

권한이 필요한 작업(파일 쓰기, 명령 실행 등)은 실행 전 승인을 요청합니다:

```
Claude가 다음 작업을 실행하려 합니다:
  Write: src/app.py

허용하시겠습니까? [y/n]
```

---

## 파일 @-mention

`@` 기호 뒤에 파일 경로를 입력하면 해당 파일을 대화 맥락에 포함시킵니다.

```
> @src/main.py 이 파일에서 버그를 찾아줘
```

```
> @package.json 의존성 목록을 정리해줘
```

```
> @README.md 이 내용을 바탕으로 설치 가이드를 다시 써줘
```

### 여러 파일 동시 참조

```
> @src/auth.py 와 @src/user.py 의 관계를 설명해줘
```

### 디렉토리 참조

```
> @src/ 폴더 구조를 설명해줘
```

> **팁:** Tab 키를 누르면 파일 경로 자동완성이 됩니다.

## 세션 저장 및 재개

Claude Code는 대화 세션을 자동으로 저장합니다.

### 최근 세션 계속하기

```bash
claude -c
```

또는 대화형 세션 내에서 새 대화를 시작할 때:

```bash
claude -c "이전 작업 이어서 로그인 기능 완성해줘"
```

### 특정 세션 재개

세션 ID 또는 이름으로 재개할 수 있습니다.

```bash
claude -r "my-login-feature"
claude -r "abc123def456"
```

### 세션 목록 확인

```bash
claude --list-sessions
```

출력 예시:
```
ID              이름                  최근 활동
abc123          my-login-feature      2025-03-23 14:30
def456          api-refactoring       2025-03-22 09:15
ghi789          (이름 없음)           2025-03-21 18:00
```

---

## 세션 이름 변경

대화형 세션 도중 `/rename` 명령으로 현재 세션의 이름을 지정합니다.

```
/rename my-login-feature
```

나중에 `-r` 플래그로 이름을 사용해 재개할 수 있습니다:

```bash
claude -r "my-login-feature"
```

---

## 세션 종료

### Ctrl+C

현재 생성 중인 응답을 중단하거나, 빈 프롬프트에서 누르면 세션을 종료합니다.

```
> (Ctrl+C)
Session saved. Goodbye!
```

### /exit 명령

```
/exit
```

---

## 대화 기록 초기화

현재 세션의 대화 기록을 지우되, 세션 자체는 유지합니다.

```
/clear
```

컨텍스트가 너무 길어져 응답 품질이 떨어질 때 유용합니다.

---

## 슬래시 명령 요약

| 명령 | 설명 |
|------|------|
| `/help` | 사용 가능한 명령 목록 |
| `/login` | 로그인 |
| `/logout` | 로그아웃 |
| `/rename <이름>` | 세션 이름 지정 |
| `/clear` | 대화 기록 초기화 |
| `/exit` | 세션 종료 |
| `/status` | 현재 상태 확인 |
| `/model` | 사용 모델 확인/변경 |

## 실습: 간단한 코드 파일 만들어보기

### 단계 1: 세션 시작

```bash
cd ~/projects/my-first-claude
claude
```

### 단계 2: 파이썬 파일 생성 요청

```
> hello.py 파일을 만들어줘. 사용자 이름을 입력받아 인사하는 간단한 프로그램이야.
```

Claude가 생성한 파일 예시:

```python
# hello.py
name = input("이름을 입력하세요: ")
print(f"안녕하세요, {name}님!")
```

### 단계 3: 파일 참조하여 수정 요청

```
> @hello.py 여기에 나이도 입력받아서 출력해줘
```

### 단계 4: 세션 이름 지정 후 종료

```
/rename hello-practice
/exit
```

### 단계 5: 세션 재개

```bash
claude -r "hello-practice"
```

```
> 이전에 만든 hello.py에 예외 처리도 추가해줘
```

## 팁

- 대화가 길어질수록 토큰 사용량이 늘어납니다. 새 주제를 시작할 때는 `/clear`를 활용하세요.
- 세션 이름을 잘 지정해 두면 나중에 특정 프로젝트 작업을 빠르게 재개할 수 있습니다.
- `@` mention은 파일 내용을 컨텍스트에 추가하므로 너무 큰 파일은 필요한 범위만 언급하는 것이 좋습니다.
